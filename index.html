Python был создан в 1991 году нидерландским программистом Гвидо ван Россумом. Основные принципы Python:

  • Машинное обучение и искусственный интеллект: TensorFlow, PyTorch

  • Автоматизация: скрипты для работы с файлами, веб-страницами, Excel, базами данных

Переменные и типы данных — это базовые понятия любого языка программирования. От того, насколько правильно студент освоит работу с переменными, зависит способность создавать программы и обрабатывать данные.

Python — язык с динамической типизацией, что означает: тип переменной определяется автоматически при присвоении значения, но при этом важно понимать, какие типы данных существуют и как их использовать.

1. Понятие переменной

Переменная — это именованное хранилище данных, к которому можно обращаться по имени.

Пример переменной в Python:

age = 20
name = "Алексей"

Здесь: age — переменная, хранящая целое число 20; name — переменная, хранящая текст "Алексей".

Особенности переменных в Python:

  • Имя переменной может содержать буквы, цифры и знак подчеркивания.
  • Имя не должно начинаться с цифры.
  • Python чувствителен к регистру: Age и age — это разные переменные.

2.2. Числа с плавающей точкой (float)

Арифметические операции в Python

Арифметические операции — это основа любой программы, связанной с вычислениями. В Python доступны базовые математические операции, которые позволяют выполнять сложные вычисления, работать с числами, создавать программы для анализа данных и автоматизации процессов.

Цель практической работы:

  • Ознакомиться с арифметическими операторами Python
  • Научиться выполнять простые и сложные математические вычисления
  • Создать простые программы, которые используют арифметику
  • На примере вычисления площади прямоугольника закрепить навыки

1. Арифметические операторы

В Python существуют следующие основные арифметические операторы:

Оператор Назначение
+ Сложение
- Вычитание
* Умножение
/ Деление (возвращает число с плавающей точкой)
// Целочисленное деление (отбрасывает дробную часть)
% Остаток от деления
** Возведение в степень

Примеры:

a = 10
b = 3
print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.3333333333333335
print(a // b) # 3
print(a % b)  # 1
print(a ** b) # 1000

2. Выполнение математических вычислений

Python позволяет комбинировать операторы и использовать скобки для изменения порядка выполнения операций.

Правила приоритетов:

  1. Скобки () выполняются первыми
  2. Возведение в степень **
  3. Умножение, деление

Встроенные функции для более сложных вычислений:

  • abs(x) — абсолютное значение
  • round(x, n) — округление до n знаков
  • pow(x, y) — возведение в степень
  • max(x, y, ...), min(x, y, ...) — максимум и минимум

Пример:

x = -7.8
y = 3.14
print(abs(x))       # 7.8
print(round(y, 1))  # 3.1
print(pow(2, 5))    # 32
print(max(5, 10, 2)) # 10

Условные операторы в Python

Условные операторы — это основной инструмент управления потоком выполнения программы. Они позволяют программе принимать решения в зависимости от значений переменных или результатов вычислений.

Python использует ключевые слова if, elif и else для создания условных конструкций.

Цель практической работы:

  • Познакомиться с оператором if
  • Изучить конструкции if - else и elif
  • Научиться использовать операторы сравнения
  • Создать программы, которые принимают решения на основе условий
  • На примере определения чётного и нечётного числа закрепить навыки

1. Оператор if

Оператор if позволяет выполнить блок кода только в том случае, если условие истинно (True).

Синтаксис:

if условие:
    блок кода

Пример:

age = int(input("Введите ваш возраст: "))
if age >= 18:
    print("Вы совершеннолетний")

Здесь блок кода выполнится только если возраст больше или равен 18. В Python отступы (обычно 4 пробела) обязательны для определения блока кода.

2. Операторы if - else

Если нужно выполнить альтернативный код в случае, когда условие ложно, используется конструкция if - else.

Синтаксис:

if условие:
    блок_кода_если_истина
else:
    блок_кода_если_ложь

Пример:

age = int(input("Введите ваш возраст: "))
if age >= 18:
    print("Вы совершеннолетний")
else:
    print("Вы несовершеннолетний")

3. Оператор elif

Если нужно проверить несколько условий, используют elif.

num = int(input("Введите число: "))
if num == 0:
    print("Число равно нулю")
elif num % 2 == 0:
    print("Число четное")
else:
    print("Число нечетное")

Циклы в Python

Циклы — это конструкции, позволяющие повторять выполнение блока кода несколько раз. Циклы широко используются в программировании для автоматизации повторяющихся действий, обработки списков, массивов, данных и генерации последовательностей.

Python поддерживает два основных типа циклов: for и while.

Цель практической работы:

  • Познакомиться с циклом for
  • Изучить цикл while
  • Научиться повторять действия в программе
  • На примере вывода чисел от 1 до 10 закрепить навыки

1. Цикл for

Цикл for используется для перебора элементов последовательности, такой как список, строка, диапазон чисел или кортеж.

Синтаксис:

for переменная in последовательность:
    блок кода

Примеры:

Вывод чисел от 1 до 5:

for i in range(1, 6):
    print(i)

Пояснение: range(1, 6) создаёт последовательность чисел: 1, 2, 3, 4, 5. Переменная i последовательно принимает значения из диапазона.

Вывод букв из строки:

word = "Python"
for letter in word:
    print(letter)

2. Цикл while

Цикл while выполняется, пока условие истинно (True).

Синтаксис:

while условие:
    блок кода

Пример (Вывод чисел от 1 до 5):

i = 1
while i <= 5:
    print(i)
    i += 1  # увеличение счетчика

Реализация вывода от 1 до 10 через while:

i = 1
while i <= 10:
    print(i)
    i += 1

5. Вложенные циклы

Циклы могут быть вложенными друг в друга, что позволяет создавать более сложные вычисления.

Пример: таблица умножения

for i in range(1, 6):
    for j in range(1, 6):
        print(f"{i}*{j}={i*j}")
    print(" ")

7. Распространённые ошибки

1. Бесконечный цикл в while:

    print(i) # i не увеличивается -> бесконечный цикл!

Правильно:
